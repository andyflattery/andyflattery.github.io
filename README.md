# andyflattery.com

Personal site and landing page for the Leo & Henry adventure series.

Live at [andyflattery.com](https://andyflattery.com).

## About

A simple static site hosted on GitHub Pages. Collects email signups for the free PDF of *The Quest for the Lost Sword*, book one of the Leo & Henry series.

## Stack

- Static HTML / CSS
- Hosted on GitHub Pages
- Email capture via Kit (ConvertKit)
- Domain managed through Namecheap
